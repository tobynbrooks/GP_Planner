# GP Weekly Predictor

A Next.js application for forecasting Gross Profit for garage businesses using a regression model.

## Features

- Real-time GP forecasting based on:
  - Bookings
  - Hours Available
  - Efficiency
  - Rate (Â£/hour)
- 87% accuracy prediction model
- Clean, responsive UI built with Tailwind CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to use the calculator.

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- React 19
